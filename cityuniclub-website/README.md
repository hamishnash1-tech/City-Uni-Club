# City University Club Website

A React + TypeScript website that mirrors the iOS app functionality.

## Features

### Authentication
- Member login with email/password
- Session persistence
- Protected routes

### Pages
- **Home** - Membership card display
- **Events** - Upcoming club events
- **News** - Club news and announcements  
- **Reciprocal Clubs** - Browse 450+ clubs worldwide
- **LOI Request** - Request Letter of Introduction

## Quick Start

```bash
cd cityuniclub-website
npm install
npm run dev
```

Open http://localhost:5173
